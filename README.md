# What this tests

This tests that JooQ `DefaultRecordMapper` is able to automatically fill nested properties into objects' constructors using `@ConstructorProperties` anntoation.

Works fine for simple properties but not for nested objects.

I cannot use setters in objects since I want all of my model objects to be immutable.
